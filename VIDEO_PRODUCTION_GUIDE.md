## üé• RECORDING TIPS

### For Screen Recordings:
1. Set browser to 1920x1080 window size
2. Hide bookmarks bar and extensions
3. Use incognito mode for clean UI
4. Record at 60fps for smooth playback
5. Add 2-3 seconds before/after each action for editing

### For Voiceover (ElevenLabs):
1. Export narration as separate audio file
2. Use "Professional Male" or "Confident Female" voice
3. Set speed to 1.1x (slightly faster than normal)
4. Add 0.5s pause between scenes in the script

### For Editing:
1. Use 1920x1080 resolution (1080p)
2. Export at 30fps (standard for web)
3. Add subtle background music (royalty-free from YouTube Audio Library)
4. Keep transitions simple (cuts and cross-dissolves)
5. Add text overlays for key points (optional but helpful)
